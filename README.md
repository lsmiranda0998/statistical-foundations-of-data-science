# Statistical Foundations of Data Science

## Overview

This repository contains the exercise lists and exams completed during the **Statistical Foundations of Data Science** graduate course at the **Federal University of Minas Gerais (UFMG)**.

The course provides the mathematical and statistical foundations required for modern Data Science and Machine Learning, covering probability theory, statistical inference, probability distributions, hypothesis testing, estimation methods, and statistical modeling.

## Technologies

* Python
* Jupyter Notebook
* Probability Theory
* Statistics
* Data Science

## Repository Contents

### Exercise Lists

The repository includes a collection of exercise lists covering the topics presented throughout the course.

Main topics include:

* Probability theory
* Random variables
* Probability distributions
* Expected value and variance
* Statistical estimation
* Confidence intervals
* Hypothesis testing
* Statistical inference

An additional notebook covering **Probability Distributions in Python** is also included.

### Exams

The repository also contains the exams completed during the course together with their corresponding statements and, when available, official solutions.

## Running the Projects

Most materials are provided as PDF documents and can be viewed directly from the repository.

Python examples can be executed using Jupyter Notebook after installing the required dependencies.

## Academic Context

This repository was developed as part of the **Statistical Foundations of Data Science** graduate course at the **Federal University of Minas Gerais (UFMG)**.

The course introduces the statistical concepts that support data analysis, machine learning, and artificial intelligence, providing both theoretical understanding and practical problem-solving through exercises and exams.
